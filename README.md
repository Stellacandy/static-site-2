Static Website on Cloud Storage

Overview:
This project demonstrates hosting a static website using cloud object storage without running or managing any servers.

Architecture
- User accesses website via HTTP
- Cloud storage serves static HTML files
- Public access configured via bucket policy


Technology Stack
- Cloud Storage (AWS S3)
- HTML
- GitHub

Setup
- Create a cloud storage bucket with S3
- Enable the static website hosting
- Upload all the HTML files/objects
- Configure public read access by creating a bucket policy that give spublic access


Key Learnings
- Difference between static and dynamic hosting
- Cloud-native alternatives to traditional web servers
- Access control basics

Cost
- Free tier only
- No running compute resources
